# Curso Vue.js da Udemy - Diário de Estudos

Este repositório é um registro do meu progresso no curso de Vue.js oferecido pela Udemy. Aqui, compartilho anotações, códigos de exemplo, e reflexões sobre os módulos estudados, servindo como um diário de bordo da minha jornada de aprendizado em Vue.js.

## Link para o curso

O curso de Vue.js da Udemy é ministrado por Jorge Sant Ana e pode ser acessado [aqui](https://www.udemy.com/course/desenvolvimento-web-avancado-com-vue-strapi-vuex-e-vuetify).

## Introdução

O objetivo deste repositório é documentar meu aprendizado e progresso no curso de Vue.js da Udemy. Cada pasta e arquivo reflete uma seção ou conceito específico abordado no curso, acompanhados de notas pessoais e exemplos de código quando aplicável.

## Utilização

Este diário é uma referência pessoal, mas sinta-se livre para explorar o conteúdo e utilizar os exemplos de código para seu próprio aprendizado em Vue.js. 

## Contribuições

Embora este seja um repositório pessoal de estudo, comentários e sugestões são bem-vindos para melhorar a qualidade e a abrangência do material.
